# 🛍️ Store Sales Analysis & Dashboard (Excel Project)

> An end-to-end data analysis project using **Microsoft Excel** to clean, process, analyze, and visualize retail sales data, generating actionable business insights.

---

## 📌 Project Overview

This project focuses on analyzing **Store sales data** to understand customer behavior, sales trends, and business performance.

The goal was to transform raw data into meaningful insights and build an **interactive dashboard** to support data-driven decision-making.

---

## 🗂 Dataset Description

The dataset contains retail sales information including:

- 👤 Customer demographics (Gender, Age)
- 🌍 Location (States)
- 🛒 Sales channels (Amazon, Flipkart, Myntra)
- 📦 Orders and purchase details

---

## 🔄 Project Workflow

### 🧹 Data Cleaning
- Removed duplicate records  
- Handled missing and inconsistent values  
- Corrected data formats (dates, text, numbers)  
- Standardized column names  

---

### ⚙️ Data Processing
- Created new columns (Age Group, Month, etc.)  
- Categorized customers for better analysis  
- Structured data for reporting and dashboarding  

---

### 📊 Data Analysis (Pivot Tables)
- Analyzed sales by:
  - Gender  
  - Age Group  
  - State  
  - Sales Channel  
- Identified key trends and performance metrics  

---

### 📈 Dashboard Creation
- Built an interactive Excel dashboard using:
  - 📊 Charts (Bar, Pie, Line)  
  - 🎛️ Slicers for filtering  
  - 📌 KPI indicators  
- Designed for easy understanding and decision-making

## 📈 Key Insights

- 👩 Women contributed approximately **65% of total sales**
- 🎯 Age group **30–49 years** contributed around **50% of sales**
- 🌍 Top-performing states:
  - Maharashtra  
  - Karnataka  
  - Uttar Pradesh  
- 🛒 Top sales channels:
  - Amazon  
  - Flipkart  
  - Myntra
